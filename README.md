### Project Overview
This project is a simple console based ordering system implemented in C++.  
It integrates customer registration with menu based ordering and employee login with order viewing functionality.

The system allows customers to register, place orders from a menu, and stores the order details in files.  
Employees can log in using file-based authentication and view all customer orders after successful login.

### Features
- Customer registration using username and password
- Menu-based ordering system
- Bill calculation based on selected items and quantity
- Order storage using text files
- Employee login using file-based authentication
- Order viewing for employees after successful login
- Unified main menu integrating all modules

### How to Run
1. Compile the C++ file:
   g++ mainproject.cpp -o ordering
2. Run the program:
   ./ordering
3. Choose options from the main menu.

### Notes
- Employee credentials are stored in employee.txt
- Orders are stored using file handling
- This project demonstrates integration of multiple modules into one system
